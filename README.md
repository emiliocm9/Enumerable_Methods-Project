# Enumerable_Methods-Project
This project was about to rebuild some Ruby methods using our own code. The challenge consisted of a rebuild of the following methods listed.

    my_each

    my_each_with_index

    my_select

    my_all?

    my_any?

    my_none?

    my_count

    my_map

    my_inject
